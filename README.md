The game is called Tower of Memes and is just a prototype for now but i plan to make a full roblox game out of this.
In the script sometimes the function names has to do something with brainrots which is caused because the game was called Tower of Brainrots before.
Game Link: https://www.roblox.com/games/101934114106955/Tower-of-Memes

## Spawn Stages
Spawns random modules from a pool. 
To add a new stage you do not need to write extra code.
Also has a timer to generate a new layout.

## Inventory
InventoryLocalScript.luau and InventoryScript.luau handle everthing that has to do with the inventory except saving. It also handles the meme upgrading.

## Inventory
VaultLocalScript.luau and VaultScript.luau handle everthing that has to do with the vault except saving. It works like the inventory but has limited space and let's you save your memes from getting deleted after a rebirth but locks it self for 30 minutes.

## Leaderstats
Leaderstats.luau handles everthing to do with player values and saving which uses the ProfileStore module. It's also the base for offline earnings.

## Offline Earnings
OfflineEarnings.luau let's you generate money with your memes offline.

## Upgrades
UpgradeHandler.luau let's you upgrade the player like more health or more offline generation.

## Player Rebirth
PlayerRebirth.luau is a rebirthing system.

## PlattformScript
PlattformScript.luau handles generating money and getting xp on the memes.

## Meme
Meme.luau is attached to every meme and is needed to pick up a meme and handles mutations.
